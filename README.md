# Overview

The Chefâ€™s App is designed to allow users to easily:

- Add new recipes
- Browse stored recipes
- Filter recipes by course type
- Remove a single recipe
- Clear all recipes
- See average prices of each course on the Home screen (Mains, Starters, Desserts)
- Navigate between screens using custom hooks instead of navigation params

## The app uses:

- Expo
- React Navigation (Stack)
- React Context API
- Custom hooks for state management
- Background images
- Scrollable recipe cards

## UI / Styling

- Consistent theme across screens
- Background images
- Rounded buttons
- Elevated white cards for recipes
- Clean layout compatible with all devices

## Running the APP

- npm install
- npx expo start

OR
## Running on Device

Scan the QR code with Expo Go
